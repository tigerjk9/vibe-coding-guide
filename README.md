## 🚀 직접 배포하기

**Netlify Drop** — 가장 쉬움
1. `index.html`을 그대로 둔 채 [app.netlify.com/drop](https://app.netlify.com/drop) 접속
2. 폴더(또는 파일)를 드래그 & 드롭 → 고정 URL 생성

**GitHub Pages**
1. 저장소 → Settings → Pages
2. Branch: `main` / `/ (root)` 선택 후 저장
